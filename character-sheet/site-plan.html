Character Sheet Site Plan

Jonathan Hobbs

WDD 131

Overview

Purpose

I want to make a more visually appealing character sheet for a table top RPG game I'm playing with friends online. It would be a place for me to easily reference all the essential details of my character including stats, level, HP, etc. in a way that is more visually pleasing than the current sheet we use.

Audience

I am the primary audience, but I could also share this template with my friends so they could fill out their own character details.

Dynamic elements

I want to be able to manage my health and other stats that will change during the course of a normal game session. I would also like a toggleable button that allows me to make edits to things like level and character stats that will change only sometimes. I'd like to add a radar chart that adjusts dynamically with my character stats if possible. In my second page I want to be able to type notes in and then add them to the page.
